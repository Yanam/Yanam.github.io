各种初始化

数组初始化

问：在java中，声明一个数组的过程中，内存是如何分配的？ 在Java中声明一个数组，jvm先在栈中给数组的引用变量分配内存空间，之后在堆中为数组对象分配内存空间。 实际上数组要初始化的是堆中的数组对象，栈内存是不用初始化的，这适用于所有引用类型。

问：数组有几种初始化方式？

  1. 静态初始化，就是指在数组定义好之后，直接为其指明元素的值。
  2. 动态初始化，只指明数组大小，元素的值为默认值。 
    //  静态初始化
int[] array = new int[]{1,2,3,4};
int[] array2 = {1,2,3,4};
//  动态初始化
int[] array3 = new int[9];

错误的例子： 

// 提供了数组初始化操作后，就不能再制定数组维度
int[] a = new int[5]{1,2,3,4,5}   //错误

// 数组常量只能在初始化时使用
int a;
a = {1,2,3,4,5}		//错误

// 没有初始化的数组不能赋值
int a[];
a[1] = 1;		//错误
