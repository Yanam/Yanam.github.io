胡诌

最近做一个异步消息的项目，涉及到序列化的模块，为了使项目得到更广泛的使用，准备支持几个常见的序列化框架的支持。由于以前只是听说过，也时常在一些技术博客上看到对各种序列化框架做的对比、测试，但从未实际使用过，所以特此记录，希望以后有也能有机会进行深入的比较分析。

  • protobuf试用 单纯的序列化框架
  • Thrift试用 RPC
  • Hessian试用 RPC
  • Kryo试用 单纯的序列化框架，原生支持Java，其他语言需要看是否有对应的版本（目前有Objective-C、Clojure、Scala）
