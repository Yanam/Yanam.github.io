Linux命令行

Bash默认为emacs编辑模式，以下大部分快捷键仅当emacs编辑模式时有效。

  • Ctrl + e 跳到行尾
  • Ctrl + a 跳到行首
  • Ctrl + u 从光标 删除 到 命令行首
  • Ctrl + k 从光标 删除 到 命令行尾
  • Ctrl + y 粘贴至光标处
  • Ctrl + c 终止命令（杀掉当前进程）
  • Ctrl + d 退出当前shell
  • Ctrl + z 将当前进程转到后台运行，使用fg命令恢复。
  • Ctrl + r 搜索之前执行过的命令（就是根据关键字搜索history）
