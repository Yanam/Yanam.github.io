docker服务启动失败

手欠把docker服务进程给kill掉了，导致执行docker命令时提示：“Cannot connect to the Docker daemon. Is 'docker -d' running on this host?”

怎么办呢？ 

# 启动一下服务
$ sudo service docker start

# 或者
$ sudo docker -d

在启动服务的时候可能会提示“/var/run/docker.pid still exists...”

把这个文件删掉在执行启动服务就行了。
