基本概念

语法

与java、c语言类型的语言规则，需要注意的几点：

  1. 区分大小写
  2. 标示符的第一个字符必须是字母、下划线"_"、美元符"$"。
  3. 注释使用c语言风格的注释，例子： 
    //单行注释

/*
*多行注释
*
* /

数据类型

5种基本数据类型：Undefined、Null、Boolean、Number、String 1种复杂数据类型：Object

typeof 操作符

用来检测数据类型，可能返回的结果有：

  1. "undefined" 值未定义
  2. "boolean" 值为布尔值
  3. "string" 值为字符串
  4. "number" 值为数值
  5. "object" 值为对象或null
  6. "function" 值为函数 
var message = "hi";
alert(typeof message); //"string"
alert(typeof 95);      //"number"

typeof 是操作符不是函数，因此括号不是必需的。

Undefined 类型

Undefined类型只有一个值，即特殊的undefined。在使用var声明但为赋值的变量值就是undefined。 

var message;
alert(message == undefined);

未赋值的变量与未定义的变量还是有区别的。例子： 

var age;		//声明未赋值
//var message   	//被注释掉，message未声明
alert(age);		//"undefined"
alert(message); 	//产生错误
alert(typeof age);	//"undefined"
alert(typeof message);  //"undefined"

但是对未声明变量执行typeof也会返回undefined

Null 类型

Null类型也只有一个值，即null，表示一个空指针对象，所以使用typeof操作符检查时返回"object"。 

var car = null;
alert(typeof car);	//"object"
//由于undefined值派生自null值
alert(null == undefined);	//true

Boolean 类型

Boolean类型的字面值区分大小写，正确的值为：true、false。所有类型都可以转换为Boolean类型，可以调用转换函数Boolean()。 各数据转换规则：
数据类型 转换为true的值 转换为false的值
Boolean true false
String 任何非空字符串 ""(空字符串)
Number 任何非零数值 0和NaN
Object 任何对象 null
Undefined n/a undefined

Number 类型

浮点数值

关于浮点数值计算会产生舍入误差的问题，试着基于IEEE754数值的浮点计算的通病。 

if(0.1+0.2==0.3){//真正结果是0.300000000000000004
	alert("You got 0.3.");
}

NaN

NaN,即非数值（Not a Number）是一个特殊的数值，这个数据用来表示应返回数值的操作未返回数值的情况。 NaN有两个特点：1）涉及NaN的操作都返回NaN。2）NaN与任何值都不相等，包括NaN本身。 isNaN()函数帮我们确定一个参数是否”不是数值“。例： 

alert(NaN==NaN);	//false
//
alert(isNaN(NaN));	//true
alert(isNaN(10));	//false
alert(isNaN("10"));	//false(可以呗转换成数值10)
alert(isNaN("blue"));	//true(不能转换成数值)
alert(isNaN(true));	//true(可以被转换成数值1)

数值转换

3个可以进行数值转换的函数：Number()、parseInt()、parseFloat()。第一个可以用于任何数据类型，后两个专门用户字符串。 [ ] 数值转换详细描述。

String 类型

ECMAScript中的字符串是不可变的。 要将一个值转换为字符串有两种方法：

  1. toString()方法，几乎所有值都有该方法，除了null和undefined。默认情况不需要参数，但是数值调用toString()时可以选择传递一个参数。例子： 
    var found = true;
alert(found.toString());	//字符串"true"
var num = 10;
alert(num.toString());		//"10"
alert(num.toString(2));		//"1010"
  2. String()函数，null和undefined也可使用。 
    var value1 = 10;
var value2 = true;
var value3 = null;
var value4;
alert(String(value1));		//"10"
alert(String(value2));		//"true"
alert(String(value3));		//"null"
alert(String(value4));		//"undefined"

Object 类型

Object 类型是所有它的实例的基础（就像java中Object对象一样）。它的每个实例都有下列属性和方法：

  1. constructor 保存着创建当前对象的函数。
  2. hasOwnProperty(propertyName) 用于检查给定的属性在当前对象实例中是否存在。
  3. isPrototypeOf(object) 用于检查传入的对象是否是另一个对象的原型。
  4. propertyIsEnumerable(propertyName) 用于检查给定的属性是否能够使用for-in语句来枚举。
  5. toString() 返回对象的字符串表示。
  6. valueOf() 返回对象的字符串、数值或者布尔值表示。通常跟toString()方法返回值相同。
