pathogen

pathogen是一个管理vim插件的工具，方便了插件的安装与删除，很好用。

安装

$ mkdir ~/.vim/
$ mkdir ~/.vim/{autoload,bundle}
$ cd ~/.vim/
$ git init 
$ curl -Sso ~/.vim/autoload/pathogen.vim \
$  https://raw.github.com/tpope/vim-pathogen/master/autoload/pathogen.vim
如果用的是window，需要把~/.vim换为~/vimfiles。

使用

在你的vimrc文件头部加上这句话execute pathogen#infect()

如果你是vim新手，直接在命令行中打vim ~/.vimrc,然后复制下面的配置到文件中。 

excute pathogen#infect()
syntax on
filetype plugin indent on

git与vim插件

安装vim插件

安装还是很简单的，只要插件在git有资源就可以。 

# 添加子模块
$ git submodule add https://github.com/vim-scripts/blackboard.vim.git bundle/color
# 初始化子模块
$ git submodule init
# 更新子模块
$ git submodule update
# 初始化submodule下的所有模块
$ git submodule foreach git submodule init
$ git submodule foreach git submodule update

删除vim插件

删除稍微麻烦一点，大概分三歩：

  1. 在命令行中执行 $ git rm --cached bundle/color
  2. 编辑.gitmodules文件和.git/config文件，将子模块的相关配置删除
  3. 删除子模块残留的目录，本例就是删除bundel/color
