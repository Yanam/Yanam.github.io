前言

Linux的发行版本大致有两类：1）商业公司维护的；2）社区组织维护的。

  • 商业维护的代表：RedHat，Centos、Fedora
    1. 安装包格式rpm，命令是：rpm -参数
    2. 包管理工具yum
  • 社区组织维护的代表：Debian，Debian、Ubuntu
    1. 安装包格式deb，命令是：dpkg -参数
    2. 包管理工具apt-get
tar包只是一种压缩文件格式，rpm相当于windows中的安装文件。

YUM

YUM(Yellow dog Updater,Modified)，开源的基于RPM的命令行包管理工具。能够从指定服务器下载RPM包并安装，并自动处理依赖关系。YUM官网

命令用法通过man可以看到，这里给出几个常用的例子。

示例： 

# 安装软件
$ yum install foo-x.x.x.rpm

# 删除软件
$ yum remove foo-x.x.x.rpm 

$ yum erase foo-x.x.x.rpm

# 升级软件
$ yum update foo 

$ yum upgrade foo

# 搜索软件
$ yum search foo

RPM

RPM RedHat Package Manager（RedHat软件包管理工具）。

本来打算列一下常用的属性，想想还是直接man一下更清楚。

常用示例： 

$ rpm -q httpd		# 查询httpd是否安装

$ rpm -ivh   /home/temp/jdk-8u25-linux-x64.rpm	# 按路径安装并显示进度

$ rpm -ivh --relocate 

$ rpm -ivh --test jdk-8u25-linux-x64.rpm	# 用来检测依赖关系，并不真正安装

$ rpm -Uvh --oldpackage jdk-8u25-linux-x64.rpm 	# 新版本降级为旧版本

$ rpm -ql httpd 	# list所有文件安装目录

$ rpm -ivh jdk-8u25-linux-x64.rpm	#安装

$ rpm -Uvh jdk-8u25-linux-x64.rpm	#升级

$ rpm -e jdk-8u25-linux-x64.rpm	#删除

dpkg

dpkg(Debian Package的简写)，它是Debian及其派生版linux的包管理器。

APT

APT(Advanced Packaging Tools)，高级包装工具，是Debian及其派生版的软件包管理器。

它有以下几个主要命令组成：

  1. apt-get
  2. apt-cache
  3. apt-file

示例： 

# 搜索
$ apt-get search xxx 

# 安装
$ apt-get install xxxx 

# 卸载
$ apt-get remove xxx 

# 升级
$ apt-get update
