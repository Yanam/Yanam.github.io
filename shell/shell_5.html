输入/输出

文件描述符(File Descriptor)：进程对其所打开文件的索引，形式上是非负整数。

类Unix系统中，常用的特殊描述符：

文件描述符 名称 常用缩写 默认值
0 标准输入 stdin 键盘
1 标准输出 stdout 屏幕
2 标准错误输出 stderr 屏幕

/dev/null：在类Unix系统中，它表示为空设备，它丢弃一切写入其中的数据（但会报告写入成功），读取它会立即得到一个EOF。

它也被称为bit bucket或者黑洞

cat命令

cat ：显示文件内容，创建文件 命令格式：cat [option] filename1....filenamen，options如下：

  • -n 由1开始对所有输出行进行编号
  • -b 与-n类似，只不过不对空白行编号
  • -s 当遇到两行以上的空白行，就替换为一行 
    # 显示名为file1 的文件
$ cat fiel1
# 显示3个文件
$ cat file1 file2 file3

tee命令

tee：在不影响原本I/O的情况下，将stdout复制到一份到档案去（读取标准输入的数据，并将其内容输出成文件）。 命令格式：tee [-ai] [文件..]

  • -a 从文件后面插入数据，而不是覆盖原有数据。
