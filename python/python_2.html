字典dict

dict函数

字典方法

clear方法
清楚字典的所有项。
copy方法
浅复制一个字典。
fromkeys方法
使用给定的键建立新的字典，每个键默认值为None。
get方法
取得字典某项的值，如果key不存在则返回None。
has_key
检查字典是否包含给定的键。与 k in dict类型。
items
将字典所有项以列表的形式返回。
iteritems
与items类型，不过返回的是迭代器。
keys和iterkeys
与上面类似，不过返回的是键。
values和itervalues
与上面类似，不过返回的是值，与其他不同的是返回值会包含重复内容。
pop方法
获得给点键的值，然后从字典中移除。
popitem方法
与上面类似，不过是随机弹出某个项。
setdefault方法
与get类似，不过还能在字典不给定键的情况下设定相应的键值。
update方法
可以用一个字典项更新另一个字典。
# dict
items = [('name','tom'),('age',12)] 
d = dict(items)
print d				#{'age':12,'name':'tom'}
# fromkeys
dict.fromkeys(['name','age'])
print dict			#{'age':None,'name':None}
