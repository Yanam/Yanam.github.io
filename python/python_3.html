其他语句

if、for、while等语句都是以冒号“:”结尾。 else if 在python中简写为elif。 循环语句可以跟else语句连接,当循环结束时执行else子句，但当循环被break中断时则不执行。 try 语句也可以后接else，在未发送异常时执行。

布尔值

标准值：False、True False、None、0、""、()、[]、{}以上这些值在作为布尔表达式时会被解释为假Flase。

range和xrange函数

range函数
range([start,] stop [,step] ) ,根据给出的start与stop指定的范围以及step设定的步长，生成一个序列。
xrange函数
与上面相同，只不过生成的不是一个数值，而是一个生成器。
>>>  range(5)
[0,1,2,3,4]
>>>  range(1,5)
[1,2,3,4]
>>>  range(0,6,2)
[0,2,4]


# xrange
>>> xrange(5)
xrange(5)
>>> list(xrange(5))
[0,1,2,3,4]

列表推导式

列表推导式（list comprehension）是利用其他列表创建新列表的一种方法。工作方式类似于for循环。 

>>> [x*x for x in range(10)]
[0,1,4,9,16,25,36,49,64,81]
>>> [x*x for x in range(10) if x%3 ==0]
[0,9,36,81]

参数

位置参数
因为它们的位置很重要。但有些时候，参数的顺序很难记住，为了让事情简单我们可以使用关键字参数。
关键字参数
最厉害的地方在于可以在函数中给参数提供默认值，当它和位置参数联合使用时，注意要不位置参数放在前面。
收集参数
收集其余位置的参数。

lambda函数

lambda函数
python从lisp借鉴过来创建匿名函数的方法。lambda函数可以接收任意多个参数（包括可选参数）并返回单个表达式的值。
lambda函数是一种编码风格问题。不一定非要使用它，任何使用它的地方都可以定义一个普通函数来替换。 
def f(x):
    return x*2
>>> f(3)
6
>>> g = lambda x:x*2
>>> g(3)
6

>>>(lambda x:x*2)(3)
6

内建函数filter、map、reduce

filter函数
接收两个参数，分别为function和sequence。该函数返回一个执行function结果为true的序列。如果序列是字符串或者元组，那么返回的结果也是，否则返回列表。
map函数
接收两个参数，分别为function和sequence。该函数返回sequence中元素执行function后的结果的列表。
reduce函数
接收两个参数，分别为function和sequence。该函数一个调用二元函数的结果的和。
def f(x):
    return x%2!=0 and x%3!=0
>>> filter(f,range(2,25))
[5,7,11,13,17,19,23]

def cub(x):
    return x*xx
>>> map(cub,range(1,11))
[1,8,27,64,125,216,343,512,729,1000]
# 当传递的参数是两个序列时，function也必须接收两个参数
def add(x,y):
    return x+y
>>> map(add,range(8),range(8))
[0,2,4,6,8,10,12,14]

def add(x,y):
    return x+y
>>> reduce(add,range(1,11))
55
