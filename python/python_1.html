字符串

两种值转换为字符串的机制：

str函数
它会把值转换为合理形式的字符串
repr函数
它以合法的python表达式的形式来标示值

print repr("hello,world")		#'hello,world'
print repr(100000L)			#100000L
print str("hello,world")		#hello，world
print str(100000L)			#100000

两种取得输入值的方式：

input函数
它会假设用户输入的是合法的python表达式
raw_input函数
它会把所有的输入当做原始数据(raw data)，然后放入字符串中

特殊字符串语法：

长字符串
如果需要一个非常长的字符串，它可能会跨行，那么使用三引号代替普通引号。

# 因为这种与众不同的引用方式，在字符串之中可以使用单引号、双引号，而不需要转义。
print '''this is a very longstring.
it  contiuse hrre
"hello ,world"
it's  not over yet
'''
原始字符
以r开头，可以放入任何字符而不会对反斜线进行转义，但是最后一个字符不能使反斜线。
print r'Let\'s go!'		#Let\' go!

序列(sequence)

python包含六种内建的序列：元组、列表、字符串、Unicode字符串、buffer对象和xrange对象，序列中的每个元素都有自己的编号。

通用序列操作

所有序列都可以进行某些特定的操作。

索引

索引操作用来访问单个元素。

序列中的所有元素都有编号，从0开始递增。使用负数索引时，最后一个元素的编号是-1。 

greeting = 'hello'
print greeting[0]		#h
print greeting[-1]		#o

分片

分片操作用来访问一定范围内的元素。

分片操作的实现需要两个索引作为边界。第一个索引元素是包含在分片内的，而第二个则不包含在分片内。 

print greeting[0:2]		#he
# 步长为2
print greeting[0:2:2]		#h

序列相加

同类型的序列才能相加。

乘法

greeting*2			#hellohello

成员资格检查

检查某个值是否在序列中，可以使用in操作符。 

'h' in greeting			#True

长度、最大值、最小值

len函数
返回序列中包含元素的数量。
min函数
返回序列中的最小元素。
max函数
返回序列中的最大元素。

列表list

列表是可变的(mutable)。 数据形式[1,2,3]

list函数

list函数
以一个序列为参数并把它转换为列表

列表方法

append方法
在列表末尾追加新的元素。
coutn方法
统计某个元素在列表中出现的次数。
extend方法
在列表的末尾一次性追加另一个序列中的多个值，也就是用新的序列扩展原有序列。
index方法
在列表中找出某个值第一次匹配项的索引位置。
insert方法
将元素插入到列表中。
pop方法
移除列表中的一个元素（默认是最后一个），并返回这个元素的值。
remove方法
移除列表中某个值的第一个匹配项。
reverse方法
将列表中的元素反向存放。
sort方法
在原位置对列表进行排序，在原位置排序意味着改变原本的列表。

# append
tempList = [1,2,3]
tempList.append(4)
print tempList			#[1,2,3,4]
# extend
tempList.extend([1,2,3])
print tempList			#[1,2,3,4,1,2,3]

tempList.insert(3,'test')
print tempList			#[1,2,3,'test',4,1,2,3]

print tempList.pop(1)		#2
print tempList 			#[1,3,'test',4,1,2]
# remove
tempList.remove('test')		
print tempList			#[1,3,4,1,2]
# reverse
tempList.reverse()
print tempList			#[2,1,4,3,1]
# sort
tempList.sort()
print tempList			#[1,1,2,3,4]

字符串

字符串是不可变的。分片赋值是不合法的。

字符串格式化

字符串的格式化使用%来实现。

字符串方法

find方法
在较长的字符串中查找子字符串，并返回子串所在位置的最左端索引。没找到则返回-1。
join方法
split方法的逆方法，用来在队列中添加元素。
lower方法
返回字符串的小写字母。
replace方法
返回某字符串的所有匹配均被替换后的字符串。
split方法
用来将字符串分隔成序列。
strip方法
返回去除两侧（不包括内部）空格的字符串。
translate方法
只处理单个字符的替换。
# find
title = 'hello,world'
print title.find('hello')		#0
# find 还可以接受可选的起点和终点参数
title.find('hello',1)			#-1
title.find('hello',1,2) 		#-1
# join
seq = ['1','2','3']
sep = "+"
print sep.join(seq)			#'1+2+3'
# lower
title = 'Hello,World'
print title.lower()			#'hello,world'
# replace
print title.repalce('o','1')		#'Hell1,W1rld'
# split
sep = '1+2+3'
print sep.split('+') 			#['1','2','3']
# strip
sep =' 312 '
print sep.strip()			#'312'
