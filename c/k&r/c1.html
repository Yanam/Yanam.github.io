胡诌

EOF：end of file 表示文件结束符，通常值为-1，因为ASCii 码是0~255没有-1的可能，所以经常被用来判断文件的结束。linux下是Ctrl+d , windows下是Ctrl+z。

字符串：从技术角度看，字符串常量就是字符数组。字符串内部表示使用了一个空字符‘\0‘作为结尾，因此，存储字符串的无力存储但愿数比括在双引号的字符数多一个。

所以字符'x'与字符串"x"是不等的，因为后者多了 * \0 *。

在所有运算符中，下面四个运算符的优先级最高：结构运算符.和->、用于函数调用的()以及用于下标的[]

有效的指针运算：相同类型指针间的赋值运算；指针同证书之间的加法或减法运算；指向相同数组中元素的两个指针间的减法或者比较运算；将指针赋值为0或者指针与0之间的比较运算。 

int x=1,y=2,z[10];
int *ip;		// ip是指向int类型的指针

ip = &x;		// ip现在指向x
y = *ip;		// y的值现在为1
*ip = 0;		// x的值现在为0
ip = &z[0];		// ip现在指向z[0]

char **argv		//char 指针的指针
int (*daytab)[13]	//int 数组的指针
int *daytab[13]		//int 指针的数组
void *comp()		//返回 void指针 的 函数
void (*comp)()		//返回 void函数 的 指针

命令行参数：调用主函数main时，它带有两个参数。 第一个参数（习惯称为argc，用于参数计数）的值表示运行程序时命令行参数的数目； 第二个参数（称为argv，用于参数向量）是一个指向字符串数组的指针，其中每个字符串对应一个参数。我们通常使用多级指针处理这些字符串。 例如： 

echo hello, world

按照C语言的约定，argv[0]的值是启动该程序的程序名，因此argc的值至少为1，如果argc为1，则说明该程序名后面没有命令行参数。另外，ANSI标准要求argv[argc]的值必须为空指针（也就是0）。
