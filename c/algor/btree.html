前言

节点的度(degree)：一个节点含有的子树的个数。

m-way search trees(多路查找树)

m-way search tree(多路查找树)：就是一颗每个节点的度都小于等于m的树。

单个节点可以这样描述： 

int m;//这是一个 3路查找树
struct mnode{
	int in_use;//实际使用的key的个数
	Key keys[m-1];
	mnode *children[m];
}

B-tree

B-tree 是一种平衡多路查找树(balanced M-way search tree)，而且M≥2。

单个节点可以这样描述： 

int m=1024;
typedef struct BTNode *PBTNode;
struct BTNode{
	int keyNum;//实际使用key的个数
	PBTNode parent;//指向父节点
	PBTNode *children;//指针向量：children[0].....children[keyNum]
	KeyType	*key;//关键字向量：key[0]....key[keyNum-1]
}
