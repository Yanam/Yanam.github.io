希尔排序

概述

希尔排序（shell sort）：也称递减增量排序，是插入排序的一种改进算法，由DL.SShell 于1959年提出而得名。

基本思想

先取一个gap1< size ，然后将数列中间隔为gap1的数放到一组，进行直接插入排序，然后取gap2<gap1重复上面的步骤，直到gap=1，也就是将所有元素放到了一个组里排序为止。

shellsort.swf

代码示例： 

void shellsort(char num[],int size){
    int gap,i,j,temp;
    while((gap/=2)>0){//设置步长
        for(i=gap;i<size;i++){//分组
            for(j=i-gap;j>0&&num[i]>num[j];j-=gap){//间隔为gap的元素进行插入排序
                    temp = num[j+gap];//加gap是因为for里面提前减了gap
                    num[j+gap]=num[i];
                    num[i]=temp;
            }
        }
    }
}
